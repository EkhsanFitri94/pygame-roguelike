# pygame-roguelike

A small roguelike dungeon crawler built with Pygame. It uses procedural dungeon generation, keyboard movement, and a scrolling camera centered on the player.

## Features

- Procedural dungeon generation using a random-walk algorithm
- Keyboard movement with WASD or arrow keys
- Scrolling camera that follows the player
- Simple tile-based map rendering

## Tech Stack

- Python
- Pygame

## Local Setup

1. Install dependencies.

```bash
pip install -r requirements.txt
```

2. Run the game.

```bash
python src/main.py
```

## Notes

- The current build is a playable prototype rather than a complete game.
- Good next additions would be enemies, collisions, a UI, and an end state.
